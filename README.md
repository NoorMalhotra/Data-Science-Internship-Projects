# Data Science Internship Projects

This repository contains the projects I completed as part of my Data Science Internship. The work focuses on applying core concepts of Machine Learning, Natural Language Processing (NLP), and Exploratory Data Analysis (EDA) on real-world datasets.

---

## 📁 Project Structure

1. **Heart Failure Prediction**
2. **SMS Spam Classification**
3. **Credit Risk EDA**

---

## 📌 Projects

### 01. Heart Failure Prediction

* Built a machine learning model to predict mortality in heart failure patients
* Performed data preprocessing and feature analysis
* Applied classification algorithms for prediction

---

### 02. SMS Spam Classification

* Developed an NLP-based model to classify SMS messages as spam or ham
* Performed text preprocessing (tokenization, cleaning, vectorization)
* Implemented classification models for accurate prediction

---

### 03. Credit Risk EDA

* Conducted exploratory data analysis on a credit risk dataset
* Analyzed patterns in loan defaults, income, and credit history
* Visualized insights using plots and heatmaps

---
