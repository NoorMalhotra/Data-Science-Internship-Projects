# Data Science Internship Projects

This repository contains the projects I completed as part of my Data Science Internship. These projects focus on applying core concepts of Machine Learning, Natural Language Processing, and Exploratory Data Analysis.

---

## Projects

### 1. Heart Failure Prediction

* Built a machine learning model to predict mortality in heart failure patients
* Performed data preprocessing and model training
* Applied classification techniques for prediction

---

### 2. SMS Spam Classification

* Developed an NLP-based model to classify SMS messages as spam or ham
* Used text preprocessing techniques such as tokenization and vectorization
* Implemented classification algorithms for accurate prediction

---

### 3. Credit Risk EDA

* Conducted exploratory data analysis on a credit risk dataset
* Analyzed patterns in loan defaults, income, and credit history
* Visualized key relationships using plots and heatmaps
